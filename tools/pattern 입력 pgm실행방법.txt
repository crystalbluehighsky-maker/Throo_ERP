app.py & index.html 입력 프로그램의 정체
이 프로그램은 **'전표 생성 가이드 등록기'**라고 보시면 됩니다.

구성 요소:

app.py (Backend): Flask 또는 FastAPI로 작성되었으며, 사용자가 입력한 문장을 AI(Gemini/OpenAI)에게 보내 **벡터 값(Embedding)**으로 변환한 뒤 DB에 저장하는 역할을 합니다.

index.html (Frontend): "어떤 문장이 들어왔을 때(예: 통장에 입금확인)", "어떤 결과(예: id 14번 패턴)"가 나와야 하는지를 입력하는 간단한 웹 화면입니다.

구동 방식 (포트 정보):

현재 메인 ERP 서버(serverlogin.py)가 8000번 포트를 사용하고 있기 때문에,

입력 도구인 app.py는 충돌을 피하기 위해 보통 5000번(Flask 기본) 또는 8080번 포트를 사용하도록 설정되어 있었을 것입니다.

# 메인 서버와 별개로 새로운 터미널에서 실행
python app.py
이후 브라우저에서 http://127.0.0.1:5000 (또는 설정된 포트)로 접속하면 입력 화면이 뜹니다.


오늘 구매1팀 회식으로 220,000원 지불 완료했어.
